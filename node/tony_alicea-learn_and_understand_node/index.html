Lecture 7: JavaScript Engines and the ECMAScript Specification

Big word: ECMAScript

The standard JavaScript is based on. How the language should work.
ECMAScript is needed a standard since there are many engines, other than V8

Big word: Javascript Engine

A program that converts Javascript code nto something the computer processor can understand.
And it should follow the ECMAScript standard on how the language should work and what features it should have.

Lecture 17: First-Class functions and function expressions

Big word: First class functions

Everything you can do with other types you can do with functions.
You can use functions like strings, numbers, etc. (i.e. pass them around, set variables equal to them, put them in arrays, and more).

Big word: Expression

A block of code that results in a value
Function expressions are possible in Javascript because functions are first class.

Big word: Invoke

It means run the function.
We also say "call" the function.

Lecture 19: Objects and object literals

Big word: Name/Value pair

The name may be defined more than once, but only can have one value in any given context.
That value may be more name/value pairs.

Big word: Object: A collection of name/value pairs

The simplest definition when talking about JavaScript.

Big word: Object literal

name/value pairs separated by commas and surrounded by curly braces
